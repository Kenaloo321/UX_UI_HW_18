My UX Skills

User Research

During user research I use many skills such as: surveys, interviews, a user persona, and affinity diagrams to help better understand the users and their needs.

User Empathy

From all the data collected during the user research process. I create an empathy map based off the user persona that was created to really tell the users story.

Qualitative Interviews

I start off with surveys to find people who would/have been users of the product. Then I create a user testing plan and ask users questions about their previous experiences to learn their pain and gain points.
